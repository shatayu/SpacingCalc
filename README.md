# SpacingCalc
A program to calculate the lowest number of VEX spacers needed to fill a distance. Uses the Coin Change algorithm (http://bit.ly/2a57MVA)

Uses the following spacers:

Spacer (Nylon) 0.125"

Spacer (Nylon) 0.250"

Spacer (Nylon) 0.375"

4.6mm spacer

8mm spacer

Steel washer (0.032")

Teflon washer (0.04")
